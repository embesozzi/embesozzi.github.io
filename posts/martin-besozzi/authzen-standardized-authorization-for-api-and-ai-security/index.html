Standardized Security for Humans and Agents

This article announces the beta release of the AuthZEN plugin for Apache APISIX, turning the API and AI gateway into an AuthZEN-compliant Policy Enforcement Point.

Why It Matters

AuthZEN addresses authorization interoperability, scale, and tight coupling by standardizing how Policy Enforcement Points communicate with Policy Decision Points.

What the Plugin Enables

Apache APISIX can externalize authorization decisions to AuthZEN-compatible PDPs such as OpenFGA for ReBAC and Cerbos for policy-based access control.
